## Instalar entorno

Primero se clona el repositorio de la rama `master`.

```bash
git clone https://github.com/Dark869/Proyect_DSW.git
```

Una vez clonado se crea un entorno virtual dentro del directorio del repositorio local.

```bash
python -m venv venv
```
